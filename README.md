Blog de Fotos
Página web sencilla para visualizar una galería de imágenes centrada.

Características
Estructura: HTML5

Colores: Fondo gris oscuro y texto naranja.

Fuente: Trebuchet MS.

Estructura de archivos
index.html: Código principal.

/imagenes: Carpeta con los archivos JPG.

Notas de uso
Las imágenes tienen un ancho fijo de 250px definido en el HTML para mantener la uniformidad. Para visualizar correctamente, los archivos de imagen deben estar en la carpeta correcta.
